# Security Policy

This project is a whitebox security review workspace. Please do not include
private target source, customer data, exploit details for live systems, or run
artifacts when reporting a project issue.

For vulnerabilities in this repository itself, open a private advisory or send a
minimal report to the maintainers with:

- affected version or commit
- impact
- reproduction steps using synthetic data
- suggested fix, if known

Only run this tool against repositories and systems you are authorized to test.
